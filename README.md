# Welcome to QUIVERHS.
*A user friendly web app based on PyQuiver for calculating isotopic effects on kinetic and equilibrium.*

QuiverHS is an open-source web-based application to calculate Kinetic Isotopic Effects (KIE) and Equilibrium Isotopic Effects (EIE) using harmonic frequencies, both Bigeleisen-Mayer equation, and thermodynamic approach. Additionally, this program directly calculates and displays the component of each approach seperately. QuiverHS is based on PyQuiver, which requires Cartesian Hessian matrices, which can be calculated using any electronic structure program.
